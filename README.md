Dieses öffentliche Repository dient als zentrale Dateiablage für das Knowledge Badge "KB-EAA". Hier befinden sich alle relevanten Unterlagen, Materialien und Ressourcen.

**Hinweis:** Nicht öffentliche und vertrauliche Unterlagen (bspw. die Prüfungsfragen) sind nicht in diesem GitHub-Projekt enthalten, sondern an einem separaten, geschützten Ablageort hinterlegt.

Weitere Informationen zu Syllabi, Prüfungsterminen, Merkblättern etc. finden Sie im [KB-BFL Wiki](../../wiki)
